# xrip-Nuker
xrip, fastest nuker on discord written in py.
## Installation
For compiled version:

- Download latest version from [releases](https://github.com/jeffhenry7/xRip-Nuker/releases/tag/v7.0)
- Set the password : 1
- Extract all the content 
- Run xRip.exe

For python version:

- Install required modules from "requirements.txt"
- Provide your bot token and channel/roles/messages content in config.json file
- Provide your proxies (optional)
- Run xRip.py
